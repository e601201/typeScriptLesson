## 学習時の環境条件
- node:  v18.20.5

## インストール方法

1. リポジトリをクローンします。

   ```bash
   git clone https://github.com/e601201/typeScriptLesson.git
   cd typeScriptLesson
   ```

2. 必要なパッケージをインストールします。

   ```bash
   npm install
   ```

3. プロジェクトを開始します。

   ```bash
   tsc
   or 
   tsc -w
   ```

## 貢献方法

1. このリポジトリをフォークします。
2. 新しいブランチを作成します。
   ```bash
   git checkout -b feature/new-lesson
   ```
3. 変更をコミットします。
   ```bash
   git commit -m "Add new lesson on advanced types"
   ```
4. プルリクエストを作成します。
